# NEOVIM

This configuration is based on the [nvchad](https://nvchad.com/) project.

If you want to adjust the keymap you can do this in the mappings.lua file.
Highlights of the custom keybinds:

- Leader + lh - LSP: show documentation
- Leader + lr - LSP: rename
- Leader + lu - LSP: show referces
- Leader + la - LSP: show code actions
- Leader + ld - LSP: go to definition

- Leader + ff - FIND: files
- Leader + fw - FIND: text in files
- Leader + fo - FIND: find previously used file

- Leader + gl - PREVIEW: Markdown PREVIEW

- Going through Search occurences: Centers the screen
- Going half page up / down: Centers the screen
