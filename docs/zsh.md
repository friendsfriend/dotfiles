# ZSH

The zsh configuration requires the following tools installed:

- powerlevel10k
- zinit
- fzf
- fzf-git
- zoxide
- eza
- bat
- nvim
- thefuck
- fd
- lazygit

All of them can be installed with your favorite package manager

The following settings are included in the zsh config:

- autocompletion
- coloring
- fuzzy finding with fzf
- show matching directories in dropdown when pressing tab
- use improved tools insted of basic ones
- proper history management
- cool theming
