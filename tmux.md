# TMUX

Tmux is configured to allow easy multiplexing within the terminal.
The Highlights of the configration are:

- Leader: Control + a
- Leader + #: Split horizontal
- Leader + +: Split vertical
- Leader + c: Create new window
- Leader + Number: Switch to window with Number
- Leader + w: Show list of windows
- Leader + s: Show list of sessions
- Leader + jklh: Move to pane (like nvim)
